使用自定义字体的三种模式：
1. 懒人模式
直接使用当前预生成的.bin字体，放置到TF卡font目录下即可（如无目录手动新建）。
2. 进阶模式
使用二进制字体生成工具.exe，选择自己喜欢的ttf字体和字号，生成.bin字体（注意字体路径不能包含中文）。
3. 高手模式
如果工具有问题（个人开发，未做兼容性测试）跑不起来，工具的python源码已提供，自己研究修改即可。
祝大家玩得愉快。

Three Modes for Using Custom Fonts:

Lazy Mode
Simply use the pre-generated .bin font file and place it in the font directory on the TF card (create the directory manually if it doesn’t exist).

Advanced Mode
Use the binary font generation tool .exe to select your preferred .ttf font and font size, then generate the .bin font (ensure the font path does not contain Chinese characters).

Expert Mode
If the tool doesn’t work (developed individually and not compatibility-tested), the Python source code of the tool is provided for you to study and modify as needed.

Enjoy your customization!
